# Quiz Creator

## Functional:
* Register new user, auth, simple validation
* Create quizzes with 4 options to choose from
![quiz creator](src/assets/github/quiz-creator.gif)
* List of existing quizzes
* Interactive quizzes
![quiz](src/assets/github/quiz.gif)

## Stack:
1. React
2. Redux
3. Axios
4. Formik, Yup
5. SCSS
6. Backend - Firebase
